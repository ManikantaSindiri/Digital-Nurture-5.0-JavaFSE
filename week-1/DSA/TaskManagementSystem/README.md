# Exercise 5: Task Management System

## Scenario
You are developing a task management system where tasks need to be added, deleted, and traversed efficiently.

## 1. Understand Linked Lists
- Learn how linked lists store elements as nodes.
- Understand the difference between singly linked lists and doubly linked lists.
- Think about how linked lists are useful when insertions and deletions are frequent.

## 2. Setup
- Create a new folder for this exercise inside the DSA section.
- Prepare a Java program to manage tasks using a linked list.
- Define the task details such as title and priority.

## 3. Implementation
- Implement operations to:
  - add a new task
  - delete a task
  - traverse all tasks
  - display task details

## 4. Analysis
- Explain why linked lists are useful for dynamic task management.
- Compare linked list operations with array-based operations.
- Discuss when a linked list is preferred over other data structures.

## Expected Learning Outcome
By completing this exercise, you will understand how linked lists support efficient insertion and deletion in dynamic systems.
