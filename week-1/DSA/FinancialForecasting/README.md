# Exercise 7: Financial Forecasting

## Scenario
You are developing a financial forecasting tool that predicts future values based on past data.

## 1. Understand Recursive Algorithms
- Learn how recursion breaks a problem into smaller similar subproblems.
- Understand base cases and recursive calls.
- Think about how recursive methods can model repeated forecasting steps.

## 2. Setup
- Create a new folder for this exercise inside the DSA section.
- Prepare a Java program that uses recursion to forecast values.
- Define a simple forecasting rule, such as each future value being derived from the previous one.

## 3. Implementation
- Implement a recursive function to:
  - calculate the next forecasted value
  - generate a series of forecasted values
  - display the results

## 4. Analysis
- Explain how recursion helps simplify repeated forecast calculations.
- Discuss the advantages and limitations of recursive solutions.
- Compare recursion with iterative approaches for forecasting.

## Expected Learning Outcome
By completing this exercise, you will understand how recursive algorithms can be applied to problems involving repeated prediction steps.
